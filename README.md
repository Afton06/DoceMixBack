## Status do Projeto: 🛠️ Em desenvolvimento

Esta é a API REST que alimenta a loja Doce Mix. Desenvolvida com Node.js, ela gerencia o banco de dados, o inventário de doces e as regras de negócio.

## 📝 Sobre o Projeto
O backend foi construído seguindo o padrão MVC para garantir que o código seja fácil de manter e escalar. Ele lida com toda a persistência de dados e regras de negócio da loja.

🛠️ Tecnologias
Ambiente: Node.js

Framework: Express

Banco de Dados: PostgreSQL / MySQL


### 2. Instalar dependências:
